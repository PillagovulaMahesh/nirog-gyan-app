##  Tech Stack

### Frontend

-  React + TypeScript
-  Tailwind CSS
-  React Router DOM

- Landing page with searchable doctor list
- Individual doctor profile page with availability
- Simple appointment booking form

- Responsive and mobile-friendly UI
- (Bonus) Dynamic backend using Express API
